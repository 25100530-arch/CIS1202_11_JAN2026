Design & Development Process

Behind the scenes of this portfolio website — from initial concept to final design.

1. Wireframing & Planning

Started with low-fidelity wireframes to map out the page structure and layout.

  • Sketched basic layouts for each page (Home, About, Works, Contact)
  • Identified key sections: header/nav, hero, content areas, footer
  • Planned responsive breakpoints for mobile, tablet, and desktop
  • Created information hierarchy for better UX

2. Visual Design & Color Palette

Developed a cohesive color scheme and typography system.

  • Primary color: Burgundy (#6B1C23) for headers and key elements
  • Accent colors: Rose/Mauve (#D4A5A5, #c48b96) for interactive elements
  • Background: Cream (#F5E6D3) with pink-beige gradients (#E8D5C4-#F0E0D4)
  • Selected typography that balances elegance with readability
  • Iterated through 3+ color palette variations before finalizing

3. HTML Structure & Semantic Markup

Built semantic, accessible HTML foundation.

  • Created 6 main pages with proper heading hierarchy (h1-h4)
  • Used semantic elements: header, nav, main, section, footer
  • Implemented portfolio project pages with structured content
  • Added ARIA labels and accessibility attributes for icons

4. CSS Styling & Layout

Implemented responsive design with CSS Flexbox and Grid.

  • Flexbox layouts for header, nav, contact sections
  • CSS Grid for works portfolio (3-column → responsive)
  • Gradient backgrounds and decorative elements for visual interest
  • Hover effects and transitions for interactivity
  • Media queries for mobile-first responsive design

5. Content Development

Populated pages with meaningful, well-structured content.

  • About page: Personal introduction with circular image and skills
  • Works page: 6 portfolio projects with detailed case studies
  • Contact page: Two-column layout with profile image and social links
  • Portfolio pages: Semantic HTML with references and project details

6. Enhancement & Polish

Fine-tuned details for a polished final product.

  • Added inline SVG social media icons (Facebook, Instagram)
  • Converted reference links to clickable anchors with proper attributes
  • Refined spacing, padding, and typography across all pages
  • Tested responsive design across different screen sizes
  • Optimized color contrast for accessibility
